# Ting Wei Chang - Personal Webpage

Welcome to my personal workspace! This project is a simple, aesthetically pleasing single-page portfolio styled after Notion's "Super Builder 2.0".

## Features
- **Clean and Minimalist Design:** Uses an off-white background with system-default fonts mimicking the Notion UI exactly.
- **Dynamic Clock:** A small block that displays your current local time in real-time.
- **Responsive Layout:** A mobile-friendly grid system for displaying portfolio items, links, and essays.
- **Pure HTML & Vanilla CSS:** Lightweight, fast, and does not require complex frameworks.

## Setup
To preview the website locally simply open `index.html` in any modern web browser.

## Customization
- **Profile Picture/Emoji:** Update the `.icon-container` in `index.html`.
- **Colors:** You can modify the CSS variables under `:root` in `style.css` to enable darker or different themes.
- **Content:** Update the `.card` classes in `index.html` to link to your real projects, essays, or socials.

## Deployment
You can host this site for free instantly using GitHub Pages. Just go to your repository Settings -> Pages -> and deploy from the `main` branch.
